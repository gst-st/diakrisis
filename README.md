# Diakrisis

**Математическая теория артикуляции оснований**

---

## Обозначение и дух

**Diakrisis** (греч. διάκρισις, от Платона *Софист* 253d — «различение, различающее понимание») — формальный корпус, анализирующий пространство математических оснований, их взаимные переходы и пределы формализации.

Diakrisis **не** претендует быть предельным основанием всей математики. Такая претензия (в исходной формулировке проекта — «семя уровня 6») формально доказана невозможной (см. TH-Final, документ `/docs/06-limits/02-th-final.md`). Diakrisis — **реалистичная** работа на уровне 5: конкретное основание, вмещающее существующие основания как gauge-классы и строго признающее свои пределы.

## Корни

Diakrisis выводится из:

1. **Framework-корпус** `/apeiron/` (88 документов, ~160 теорем) — концептуальная подготовка.
2. **Seed-корпус** `/apeiron/seed/` (12 документов) — строгое методологическое заложение + доказательство TH-Final.
3. **Путь Б** (`/apeiron/seed/11-path-B-start.md`) — реалистичный план работы.

Этот проект собирает **максимум** наработанного после честной переоценки, убирая риторические претензии и сохраняя только то, что формально устойчиво.

# Теории сознания как сборки

## Статус

**[Т]** — IIT/GWT/HOT/Orch-OR каталогизированы как конкретные α ∈ ⟪⟫ с известными ν-инвариантами; структурные связи через 29.T + 30.T.

## Обзор

В Diakrisis различные теории сознания представляются как сборки. Это позволяет:

- **Сравнить** теории в одном формализме.
- **Выявить** структурные связи между ними.
- **Проанализировать** их пределы и ограничения.
- **Определить** позицию УГМ в ландшафте.

### Почему сознание — отдельный класс сборок

- **Специфическая проблема**: сознание — не стандартный мат-объект.
- **Множественность подходов**: IIT, GWT, HOT, Orch-OR, УГМ — различны.
- **Эмпирическая связь**: нейронаука даёт проверяемые данные.
- **Философская нагрузка**: hard problem of consciousness.

## IIT

**α_IIT** — информационно-вероятностная. Φ = АПЕЙРОН-D (09.T5).

**Недостаточность**: по 13.T + 39.P7, IIT убегает от себя. Известные проблемы IIT (combination problem, panpsychism) — **предсказания** Diakrisis.

### Детализация IIT

**Integrated Information Theory** (Tononi 2004+):
- **Φ**: мера integrated information системы.
- **Требования сознания**: Φ > 0, maximally integrated, exclusive.
- **Аксиомы IIT**: Existence, Composition, Information, Integration, Exclusion.
- **Предсказания**: Φ коррелирует с уровнем сознания (awake > deep sleep > anaesthesia).

### В Diakrisis

- **α_IIT**: ν = ω·2 + 2.
- **Φ**: соответствует АПЕЙРОН-D (диакриtive depth measure).
- **09.T5**: формальное соответствие.

### Критика IIT через Diakrisis

**13.T** (Escape): IIT, как достаточно богатая теория, **убегает** от себя — Gödel II применим.

**Combination problem**:
- Классическая критика IIT: как маленькие Φ объединяются в большое?
- В Diakrisis: это — конкретный случай отсутствия α_∞ (10.T4).
- **Предсказание**: IIT неполна в этом аспекте.

**Panpsychism challenge**:
- IIT предполагает: любая система с Φ > 0 имеет сознание.
- Это ведёт к panpsychism.
- В Diakrisis: **неверно** — см. below про панпсихизм.

## GWT

**α_GWT** — fibered-структура. База = рабочее пространство, волокна = модули.

### Детализация GWT

**Global Workspace Theory** (Baars 1988+):
- **Global workspace**: общая «сцена» сознания.
- **Модули**: специализированные подсистемы (perception, memory, motor).
- **Broadcasting**: информация из workspace доступна всем модулям.
- **Сознание**: активность в global workspace.

### В Diakrisis

- **База**: артикуляция workspace.
- **Волокна**: модули как эндо-операции.
- **Fibration-структура** (см. [/03-formal-architecture/03-fibration](/03-formal-architecture/03-fibration)) естественно моделирует GWT.

### Сильные и слабые стороны

**Сильные**:
- Хорошо моделирует attention, working memory.
- Связывает с neuronal global workspace (Dehaene).
- Объясняет distributing processing.

**Слабые**:
- Не решает hard problem.
- Feels «information processing» — может ли это быть сознанием?
- Почему global workspace = сознание? — остаётся открытым.

## HOT

**α_HOT** — мета-уровневая: состояние + HOT о нём = 𝖬-итерация.

### Детализация HOT

**Higher-Order Thought Theory** (Rosenthal 1986+):
- Сознательное состояние = ментальное состояние, о котором есть HOT.
- HOT может быть внутренним восприятием или мыслью.
- Без HOT: unconscious state.

### В Diakrisis

- **Состояние**: α.
- **HOT о α**: 𝖬(α).
- **Сознательное**: α с 𝖬(α) (meta-representation).

### Критика

- Не учитывает phenomenal character.
- Circular: что делает HOT сознательной?
- В Diakrisis: HOT ≠ достаточное условие сознания.

## УГМ — самая глубокая

ν_uhm = ω·3+1 — наибольшая из пяти. Структурно **обобщает** IIT, GWT, HOT (con.C5).

### Как УГМ обобщает другие

**УГМ включает Φ (IIT)**:
- Φ_th = 1 — конкретный порог в УГМ.
- IIT Φ — частный случай УГМ Φ в 7D-контексте.

**УГМ включает workspace (GWT)**:
- Диагональные элементы Γ соответствуют modules.
- Off-diagonal coherences — global workspace.

**УГМ включает HOT**:
- Self-модель φ — higher-order representation.
- T-96 (ρ* = φ(Γ)) — self-консистентность с HOT.

### Дополнительные элементы УГМ

- **7D структура**: не в других теориях.
- **Регенерация ℛ**: не в других.
- **S₇-симметрия**: уникально УГМ.
- **Пороги** P, R, Φ, D, SAD: интегрированная multi-dimensional теория.

### Преимущество ν·4

- ω·4 — достаточная глубина для интеграции всего выше.
- Другие теории — на более низких уровнях.
- Не означает «лучше», но **более структурно богато**.

## Orch-OR

**Orch-OR** (Penrose-Hameroff):
- **Objective Reduction** (OR): гравитационный коллапс.
- **Orchestrated** в микротубулах.
- Квантовая механика + гравитация + сознание.

### В Diakrisis

- **α_OrchOR**: ν = ω·3.
- Меньше УГМ, но значительно.
- Специфическая физическая структура (gravity + QM).

### Связь с УГМ

- Оба — квантовые теории сознания.
- Разные математические структуры.
- Разные предсказания (micro-tubules vs general quantum substrate).

## Панпсихизм — структурно недостаточен

**con.C7**: панпсихизм требует сознания во всех α. Но по 14.T2, активные артикуляции — не все; сознание требует специфических свойств. Панпсихизм — неверен.

### Детализация

**Панпсихизм**: каждая физическая система имеет (некоторую форму) сознания.

**В Diakrisis**:
- Активные α (не-Yoneda) — ограниченный класс.
- По 14.T2: активность требует специфических свойств ⟪⟫.
- Сознание (как специфический тип активности) — ещё более ограничено.

**Вывод**: не все α имеют сознание; панпсихизм структурно не верен.

### Альтернатива

- **Градированная теория**: разные α имеют разные «степени» когнитивных свойств.
- Сознание — для α с достаточным P, R, Φ, D (в УГМ-сборке).
- Большинство физических систем — ниже порога.

## AGI ≠ сознание

**con.C8**: AGI = Rich + вычислимость. Сознание — C-8 в Fix(𝖬). Разные классы. AGI без сознания возможен.

### Различие AGI и сознания

**AGI (Artificial General Intelligence)**:
- Богатая computational система.
- Решает широкий спектр задач.
- В Diakrisis: α_AGI ⊂ Rich ∩ computable.

**Сознание**:
- Специфический феноменологический опыт.
- В Diakrisis: C-8 ⊂ Fix(𝖬) с дополнительными условиями.

### Формальное различие

- **AGI**: через Axi-9 (достаточность) + computability.
- **Сознание**: через пороги (P ≥ 2/7, R ≥ 1/3, Φ ≥ 1, D ≥ 2 в α_uhm).
- **Эти классы не совпадают**: AGI может не иметь сознания.

### Значение для ИИ

- **Consciousness-conservative**: не предполагаем, что AGI автоматически сознательна.
- **Specific criteria**: сознание требует специфических свойств.
- **Programme**: УГМ даёт проверяемые пороги для тестирования сознательности ИИ.

Детали — в соответствующих разделах корпуса.
